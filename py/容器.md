# 操作符

| 操作/    |                                   | summary                                  |
|:-------|:---------------------------------:|:----------------------------------------:|
| 切片<br> | seq[n1,n2]<br>seq[n1,n2,step]<br> | 不会越界                                     |
| 索引     | seq[n]                            | 0&nbsp; n-1<br>-n&nbsp; &nbsp;-1<br>注意越界 |
| 重复     | seq * n                           | copy,效率不高<br>                            |
| 拼接     | seq + seq                         | copy,效率不高                                |
| 关系     | in&nbsp;<br>not in<br>            |                                          |
